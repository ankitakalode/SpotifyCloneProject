# Spotify Clone (HTML and CSS)

This is a simple Spotify UI clone created using HTML and CSS as a learning-based frontend project.

## About the Project.
The goal of this project was to practice frontend fundamentals and understand how real-world music application interfaces are structured. 
By building this project, I foucsed on layout design, styling, and improving my UI understanding.

## Technologies used 
- HTML5
- CSS3

## Features
- Spotify-inspired user interface
- Sidebar navigation layout 
- Music player section (UI only)
- Clean and structured HTML 
- CSS Flexbox and positioning

## What I Learned
- Structuring HTML for multi-section layouts
- Using CSS Flexbox for alignment and layout
- Styling components similar to real applications
- Improving spacing, fonts, and UI consistency

## Project Status
This is a frontend-only project created for learning purposes.
No backend or functionality is implemented.
